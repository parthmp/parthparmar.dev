# Parth Parmar - Portfolio

Personal portfolio website showcasing my work, projects, and experience as a Full-Stack Developer.

**Live Site:** [parthparmar.dev](https://parthparmar.dev)

## Overview

This portfolio site is built with **Nuxt 4 + Vue 3** using SSG (Static Site Generation). It showcases reusable component architecture, composable patterns, dynamic routing, and optimized image delivery with NuxtPicture - all wrapped in thoughtful UI/UX with excellent performance. **The code itself is a portfolio piece.**

## Installation & Setup

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or pnpm

### Clone & Install

```bash
# Clone the repository
git clone https://github.com/parthmp/parthparmar.dev
cd portfolio-site

# Install dependencies
npm install
```

### Development

```bash
# Start development server at http://localhost:3000
npm run dev
```

### Build for Production

```bash
# Generate static site
npm run generate
```

Reusable components designed to manage and display content efficiently across multiple portfolio pages.
